Connectors

An email inbox (IMAP) or calendar (ICS) Zade can read from. Clicking "Queue Sync" doesn't import anything by itself — it just queues a work item that still needs your approval before Zade actually reads the account.
+ Connect an inbox or calendar
Never put an actual password or token in the config below. Instead, name the environment variable that holds it — e.g. "password_env": "IMAP_PASSWORD" — and set that variable outside this UI. Zade only ever sees the variable name, never the secret itself.

Staged Items

Emails and calendar events Zade found during a sync land here as candidates first — nothing is used automatically. Import the useful ones so Zade can reason about them as evidence; dismiss the ones that don't matter.
